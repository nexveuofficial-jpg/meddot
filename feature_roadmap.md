# Meddot v2 Feature Roadmap

## Core Focus: Scalability & Engagement
The next phase of development should focus on transforming Meddot from a functional utility into a thriving community platform.

### High Priority
1.  **Search & Filtering System**
    *   **Goal**: Allow students to find notes instantly.
    *   **Features**: Filter by Subject, Year, and Topic tags. Search bar for title/content.

2.  **Real File Uploads (PDF Viewer Integration)**
    *   **Goal**: Replace mock content with actual PDF rendering.
    *   **Tech**: Integrate `react-pdf` to display uploaded files within the Notes Viewer.

3.  **Bookmarked Notes List**
    *   **Goal**: Easy access to saved content.
    *   **Feature**: A dedicated "My Library" section in the dashboard showing all bookmarked items.

### Medium Priority
4.  **Admin Analytics Dashboard**
    *   **Goal**: Insight into platform usage.
    *   **Metrics**: Total views per note, active users, most popular subjects.

5.  **Community Study Rooms**
    *   **Goal**: Real-time collaboration.
    *   **Feature**: Text-based chat rooms for each subject/year.

6.  **Progress Tracking**
    *   **Goal**: Gamification.
    *   **Feature**: "Streak" counter for consecutive days studied in Focus Mode.
